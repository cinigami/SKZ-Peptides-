# SKZ Peptipes - Project Brief

## Overview
E-commerce platform for peptides with integrated marketing automation.

## Technical Stack
- **Frontend**: React + Vite + Tailwind CSS
- **Backend**: Node.js + Express
- **Database**: PostgreSQL (product catalog, orders, customers)
- **E-commerce**: Custom solution or integration (Stripe for payments)
- **Hosting**: TBD

## Marketing Strategy
**Primary Channel**: Meta Threads
- **Soft Selling**: Educational content, health benefits, lifestyle integration
- **Hard Selling**: Direct product promotions, limited offers, testimonials

## Deliverables
1. **Website**: Full e-commerce platform
   - Product catalog
   - Shopping cart
   - User accounts
   - Payment processing
   - Order management
   - Admin dashboard

2. **Marketing Automation**:
   - Social media scheduler for Threads
   - Content calendar management
   - Analytics & reporting

3. **Content Strategy**:
   - Soft sell content templates
   - Hard sell campaign structures
   - Engagement workflows
